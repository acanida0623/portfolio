Digital Photo Cloud

Digital Photo Cloud is a Python web application I designed using Django web framework. The backend uses a PostgreSQL database as well as Amazon's S3 database API. I built the front end user interface of the application using React JS, which is a Javascript library. React is a great tool that allows the user to render all of the HTML used in the application through Javascript. React creates its own virtual representation of the application's DOM, which in turn allows for more control over specific components and quicker manipulation of the elements on the page.

Javascript Blackjack

Since I am a huge fan of Blackjack, I created this Blackjack web application, once again using React JS for the front-end interface. This was my first attempt at teaching myself React, and most of the games functionality/rules are implemented through methods inside of JS objects created with constructors.I have since learned how to use methods inside of React components entirely to build an application, which is what the designers of the library intended. I created all of the graphics using Adobe Photoshop. The animation is done with CSS transitions. React once again made it fairly easy to quickly change and control multiple elements on the app such as the cards and chips without having to reload the entire DOM.
